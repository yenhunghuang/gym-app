# GymApp CLAUDE.md

## 專案概覽

-   名稱：GymApp
-   功能：記錄自身及同訓伙伴的動作與重量
-   技術棧：Node.js + Express、React、SQLite

## 開發規範

-   程式風格：ESLint + Prettier
-   Branch 流程：feature/<task> → PR → main
-   測試：Jest 單元測試 + Supertest API 測試

## 常用命令

-   `npm run dev`：啟動開發伺服器
-   `/generate-prp INITIAL.md`：生成實作藍圖
-   `/execute-prp PRPs/*.md`：執行並測試藍圖
